# Workspace

## Overview

pnpm workspace monorepo using TypeScript. Each package manages its own dependencies.

## Stack

- **Monorepo tool**: pnpm workspaces
- **Node.js version**: 24
- **Package manager**: pnpm
- **TypeScript version**: 5.9
- **API framework**: Express 5
- **Database**: PostgreSQL + Drizzle ORM
- **Validation**: Zod (`zod/v4`), `drizzle-zod`
- **API codegen**: Orval (from OpenAPI spec)
- **Build**: esbuild (CJS bundle)

## Artifacts

### Mermaid Theme Builder (`artifacts/mermaid-theme-builder`)

- **Type**: react-vite, static frontend-only (no backend, no login, no API calls)
- **Preview path**: `/`
- **Tech**: React, Vite, Tailwind CSS, Mermaid.js
- **Purpose**: Visual theme builder for Mermaid diagrams generated by humans or AI

**Features (V0.1 — reoriented):**
- Paste Mermaid code → auto-detects diagram family (15 types supported)
- **3 first-party OKHP3 brand presets**: OverKill Hill P³, AskJamie, Glee-fully (palettes.ts)
- **4 utility/example presets**: Ocean Depth, Forest Sage, Slate Ember, Violet Mist
- UI splits palette picker into Brand Presets (with OKHP3 badge) and Theme Presets sections
- Theme Details panel shows brand family, themeIntent, version, source URL
- Extended palette schema: id, name, brandFamily, isBrandPreset, themeIntent, sourceUrls, version, attribution
- Two-way live color editor with palette-switch sync fix
- Side-by-side live preview (Original vs Themed tabs)
- Generates `%%{init:...}%%` directive-based themed output
- **Attribution system**: metadata comments toggle (default ON) + attribution badge toggle (flowchart only)
- **Custom theme naming**: name field in Export Settings, propagates to all exports
- **Rich Markdown Bootstrap**: brand metadata, usage instructions, renderer warnings, disclaimer
- **Rich Prompt Scaffold**: brand-specific guidance, required metadata comment block, strict rules
- Render-safety warnings (existing init directives, non-printable chars, long labels)
- Documentation: README.md, docs/BRAND_PRESETS.md, docs/ATTRIBUTION.md, docs/THEME_METADATA.md
- GitHub Pages compatible (pure static build)

**Key files:**
- `src/lib/palettes.ts` — palette definitions
- `src/lib/detector.ts` — diagram family detection
- `src/lib/themeEngine.ts` — theme application and export generation
- `src/components/MermaidPreview.tsx` — live Mermaid renderer
- `src/components/ColorSwatch.tsx` — two-way color editor
- `src/pages/ThemeBuilder.tsx` — main UI

### API Server (`artifacts/api-server`)
- Express 5 backend
- Shared by other artifacts that need a backend

## Key Commands

- `pnpm run typecheck` — full typecheck across all packages
- `pnpm run build` — typecheck + build all packages
- `pnpm --filter @workspace/api-spec run codegen` — regenerate API hooks and Zod schemas from OpenAPI spec
- `pnpm --filter @workspace/db run push` — push DB schema changes (dev only)
- `pnpm --filter @workspace/api-server run dev` — run API server locally

See the `pnpm-workspace` skill for workspace structure, TypeScript setup, and package details.
