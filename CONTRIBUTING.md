# Contributing to Mermaid Theme Builder

## Quick Start

1. Fork the repo
2. `pnpm install`
3. `pnpm dev`
4. Make your changes
5. `pnpm build` (verify it compiles)
6. Submit a PR

## Brand Rules (Non-Negotiable)

See AGENTS.md. Do NOT use BFS hex values or employer references.

## What We Accept

- Bug fixes
- New generic palette presets
- New example diagrams
- Accessibility improvements
- Documentation improvements

## What We Don't Accept

- Server-side features
- Third-party brand presets
- AI API integrations
