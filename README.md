# Mermaid Theme Builder

  Visual theme builder for AI-generated Mermaid diagrams.

  - Paste Mermaid code, detect diagram family, apply theme palettes
  - 4 built-in palettes: Ocean Depth, Forest Sage, Slate Ember, Violet Mist
  - Live preview, two-way color editor, styled code export
  - Markdown bootstrap and AI prompt scaffold exports
  - 100% client-side — no backend, no login, no API calls

  Built with React + Vite + Mermaid.js.
