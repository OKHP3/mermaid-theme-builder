**2. **3. The monorepo structure is overengineered for v0.1.** A pnpm workspace with `lib/` and `artifacts/` directories is enterprise-grade packaging for what's currently a single-page tool. Worth keeping if you anticipate publishing `lib/` as a standalone npm package later; worth flattening if you don't.
