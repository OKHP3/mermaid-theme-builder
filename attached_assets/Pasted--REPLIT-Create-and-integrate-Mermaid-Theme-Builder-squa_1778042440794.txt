## Icon concept

Create a square icon with this design direction:

**Core metaphor:**  
A Mermaid-style node/graph symbol being “tuned” by a theme/palette control system.

**Visual elements:**
- Square canvas with rounded corners.
- Dark OverKill Hill-style background.
- Small connected diagram nodes or flowchart blocks.
- A palette/theme control motif, such as:
  - three small swatches,
  - a color slider,
  - or a small “theme dial.”
- Subtle precision/engineering cue:
  - grid line,
  - bracket,
  - alignment mark,
  - or P³-style protocol frame.
- Avoid clutter. Icon must read at 32px and 64px.

## Recommended visual style

Use an OverKill Hill-aligned palette:

- Background: deep espresso / charcoal / near-black
- Primary accent: rust-orange / copper
- Secondary accent: teal / cyan
- Highlight: warm cream / paper white
- Optional muted grid: low-opacity gray

The icon should feel:
- technical,
- precise,
- premium,
- diagrammatic,
- not playful,
- not corporate-generic,
- not rainbow-colored.
